## Implementation

### Step 1: Generate Image via Imagen 3

```python
import vertexai
from vertexai.preview.vision_models import ImageGenerationModel

def generate_image(image_prompt: str) -> bytes:
    model = ImageGenerationModel.from_pretrained("imagen-3.0-generate-001")
    images = model.generate_images(
        prompt=image_prompt,
        number_of_images=1,
        aspect_ratio="1:1"  # square for Instagram, 4:3 for LinkedIn
    )
    return images[0]._image_bytes
```

**Note:** Imagen 3 requires Vertex AI, not just the Gemini API key.  
Set up Vertex AI project and enable the API.

### Step 2: Gemini Interleaved Output (text + image in one stream)

Use Gemini 2.0 Flash's multimodal generation to produce the final post as an interleaved stream. This is what qualifies as the Creative Storyteller requirement.

```python
import google.generativeai as genai

def generate_interleaved_post(content_draft: dict, image_bytes: bytes) -> str:
    model = genai.GenerativeModel("gemini-2.0-flash")

    prompt = f"""
    Create a final social media post combining this text and the provided image context.
    
    Post text: {content_draft['post_text']}
    Platform: {content_draft['platform']}
    Hashtags: {' '.join(content_draft['hashtags'])}
    
    Format the output as a complete, ready-to-publish post.
    """

    response = model.generate_content([
        prompt,
        {"mime_type": "image/png", "data": image_bytes}
    ], stream=True)

    return response
```

### Step 3: Post Formatter

Format based on platform:

```python
def format_post(content_draft: dict, image) -> dict:
    platform = content_draft["platform"]

    if platform == "linkedin":
        return {
            "header": content_draft["topic"].upper(),
            "body": content_draft["post_text"],
            "hashtags": " ".join(content_draft["hashtags"]),
            "image": image,
            "sources": content_draft["sources"]
        }
    elif platform == "instagram":
        return {
            "caption": content_draft["post_text"] + "\n\n" + " ".join(content_draft["hashtags"]),
            "image": image
        }
```

## Demo UI (Streamlit)

File: `app.py`

The Streamlit UI is the demo surface for judges. It must show:

1. **Input panel** — topic selector (dropdown of core topics from persona config)
2. **Pipeline status** — live progress: "Retrieving NYC data... → Generating content... → Creating image..."
3. **Output panel** — rendered post with image side by side
4. **Source panel** — expandable list of NYC Open Data sources used (proves grounding)
5. **Architecture tab** — embed the architecture diagram image for judges

```python
import streamlit as st

st.title("NYC International Student AI Influencer")

topic = st.selectbox("Select a topic", [
    "Housing costs for students in NYC",
    "Visa work restrictions (OPT/CPT)",
    "CUNY enrollment trends",
    "Best neighborhoods for international students",
    "NYC job market for graduates"
])

if st.button("Generate Post"):
    with st.spinner("Retrieving NYC data..."):
        retrieval_pack = agent1.retrieve(topic)
    with st.spinner("Generating content..."):
        content_draft = agent2.run(retrieval_pack)
    with st.spinner("Creating image..."):
        image = creative_storyteller.generate(content_draft)

    col1, col2 = st.columns(2)
    with col1:
        st.image(image, caption="Generated by Imagen 3")
    with col2:
        st.markdown(content_draft["post_text"])
        st.write(" ".join(content_draft["hashtags"]))

    with st.expander("Sources (NYC Open Data)"):
        for s in content_draft["sources"]:
            st.write(s)
```
